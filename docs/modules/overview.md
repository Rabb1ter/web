# 核心模块

本页聚合 CISO 最常用的安全治理与运营模块，后续将持续补充方法论、模板与案例。

## 安全治理体系
- 安全战略与路线图
- 组织结构与职责边界
- 指标与董事会汇报框架

## 安全运营体系
- SOC 运行机制
- 威胁检测与响应流程
- 运营指标与持续改进

## 漏洞管理体系
- 资产盘点与暴露面管理
- 漏洞情报与优先级策略
- 修复跟踪与复盘机制

## 账号权限体系
- 身份治理与权限生命周期
- 特权访问管理（PAM）
- 最小权限与访问审计

## 合规与等保体系
- 等保 2.0 要求拆解
- ISO 27001 / SOC 2 控制映射
- 证据留存与审计准备

## 应急响应体系
- 事件分级与指挥体系
- 战情室组织与沟通矩阵
- 桌面演练与复盘模板

## 汇报模板库
- 董事会安全报告
- 月度/季度运营复盘
- 风险评估与预算申请

## CISO 智能体决策规则库
- 风险优先级与处置建议
- 资源与预算分配规则
- 治理与合规策略建议
