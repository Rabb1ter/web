# 关于本站

**CISO 智能体 · 安全最佳实践知识库** 面向信息安全负责人提供权威知识体系，支持人工查阅与 AI 智能体决策。

## 核心目标
- 提供可浏览、可搜索、可引用的安全知识体系
- 为安全负责人、运维、开发、合规人员提供标准指南
- 为 CISO 智能体集群提供决策依据、SOP、模板与规则
- 内容自动更新，保持行业最新最佳实践

## 技术栈
- **知识库**：Markdown + 向量库（规划）
- **能力**：浏览、搜索、分类、AI 问答
- **自动化**：知识采集、去重、清洗、格式化、更新日志

> 当前为官网版信息架构与展示，后续将接入自动采集与 RAG 检索能力。
